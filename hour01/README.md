### 2️⃣ Binary vs Decimal vs Unary

Computers understand **binary (base 2)**.

Decimal (base 10):
123 = 1×10² + 2×10¹ + 3×10⁰

Binary example:
65 (decimal) = 01000001 (binary)

Binary digit = **bit**
0 = off
1 = on

8 bits = 1 byte

---

### 3️⃣ ASCII & Unicode

Character 'A' → 65 → 01000001  
Character 'a' → 97  

Uppercase to lowercase difference = 32

Unicode = superset of ASCII.

---

### 4️⃣ RGB Color Representation

Each pixel:
- 8 bits Red
- 8 bits Green
- 8 bits Blue

Range: 0–255

White:
(255, 255, 255)

Each pixel = 3 bytes

---

### 5️⃣ Algorithm & Big-O

Binary Search idea:
Instead of tearing every page in a phonebook,
divide in half each time.

Time Complexity:
- Linear → n
- Quadratic → n²
- Logarithmic → log₂n
